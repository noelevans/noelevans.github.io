While more flexible, using lambdas in map/filter calls are slower than using the operator library of pre-built functions. Some useful examples:

>>> import operator
>>> map(operator.itemgetter(0), [(1, ), (2, ), (3, )])
[1, 2, 3]

>>> filter(operator.truth, [1, 0, 7])
[1, 7]

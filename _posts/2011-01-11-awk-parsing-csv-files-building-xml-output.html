This awk command takes an input CSV file (delimited by a comma set with the -F flag) and for each row, prints an XML tage <elementName  id="?"  name="????"/>

awk -F , '{print "<elementName  id=\""$1"\"  name=\""$2"\"/>"}'  input.csv
